# GCP Architect Hub

This project is a SvelteKit-based documentation website tailored to showcase Google Cloud Platform (GCP) offerings and the Google Cloud Well-Architected Framework, specifically designed to help study for the GCP Architect Exam.

## Features
- **Comprehensive Index**: Includes 191 GCP services extracted from the provided documentation.
- **Modern UI**: Built with SvelteKit and Tailwind CSS for a sleek, responsive, and fast experience.
- **Well-Architected Framework**: Dedicated section outlining core solution design principles.

## Getting Started

### Prerequisites
- Node.js (v18 or newer recommended)
- npm or pnpm

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
To start the development server:
```bash
npm run dev -- --open
```
The site will be available at `http://localhost:5173`.

### Building for Production
To create a production-ready build:
```bash
npm run build
```
You can preview the production build with `npm run preview`.
